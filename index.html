Proyectos

Proyecto 1: Sitio Web de Comercio Electrónico

proyecto consiste en un sitio web de comercio electrónico desarrollado con HTML, CSS y JavaScript. La plataforma permite a los usuarios navegar por una variedad de productos, añadirlos al carrito de compras, y completar la compra mediante una pasarela de pago integrada. Se utilizó Bootstrap para el diseño responsivo y JavaScript para la gestión de la lógica del carrito de compras.

Proyecto 2: Aplicación de Tareas

Desarrollé una aplicación web para la gestión de tareas utilizando React.js para la interfaz de usuario y Firebase para la autenticación y almacenamiento de datos. La aplicación permite a los usuarios registrarse, añadir, editar y eliminar tareas, así como marcar tareas completadas. También incluye funcionalidades de filtrado por estado y fechas de vencimiento.
